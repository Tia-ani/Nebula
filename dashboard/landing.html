01 — SUBMIT
Developer sends tasks
Array of prompts via API. Each becomes a separate encrypted chunk.
02 — ENCRYPT
AES-256 per session
Master creates a unique key per worker. Destroyed on disconnect.
03 — DISTRIBUTE
Chunks fan out
Idle devices process in parallel. Worker dies — chunk auto-reassigns.
04 — RETURN
Results assembled
Back to developer in order. Contributor earns credits per task.

Why Nebula

Built different.

Not another OpenAI wrapper. Actually distributed. Actually cheap. Actually yours.

10× cheaper
Idle compute costs nothing. We pass that to developers. Fraction of AWS price, always.
Per-session encryption
AES-256 key per worker, destroyed on disconnect. Compromise one node — zero exposure.
Fault tolerant
Worker dies mid-job? Chunk auto-reassigns to the next available node. Jobs always complete.
Any device
Node worker for heavy AI. Browser worker coming — open a URL and contribute.
Three lines of code
SDK hides all complexity. Chunking, encryption, fault tolerance — invisible to the developer.
Built for India first
Targeting Indian AI startups and researchers. UPI cashout for contributors on the roadmap.
